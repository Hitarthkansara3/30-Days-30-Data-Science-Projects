## 📌 Project Overview

The COVID-19 pandemic impacted millions of lives worldwide. Understanding trends in infections, recoveries, and deaths is essential for data-driven decision making.

This project performs **Exploratory Data Analysis (EDA)** on COVID-19 datasets to uncover global trends, compare countries, and generate meaningful insights using Python.

---

## 🎯 Objectives

- Analyze country-wise COVID-19 cases, recoveries, and deaths  
- Identify the most affected countries  
- Compare COVID growth trends across countries  
- Study daily global case progression  
- Calculate recovery rate and death rate  
- Perform USA county-level analysis  
- Visualize patterns using professional charts  

---

This project uses the following CSV files:

## 💡 Key Insights

- Certain countries experienced significantly higher confirmed cases than others.  
- Recovery rates varied widely across regions.  
- Death rates were higher in some countries despite lower total cases.  
- Global case counts increased in waves over time.  
- Some USA counties became major hotspots during the pandemic.
