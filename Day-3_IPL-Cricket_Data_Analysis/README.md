# 🏏 IPL Cricket Analytics Dashboard

An interactive **Power BI Dashboard** built using IPL ball-by-ball cricket data to analyze player performance, team statistics, match trends, and dismissal insights. This project transforms raw cricket data into meaningful visual insights for better understanding of the game.

---

## 📌 Project Overview

The Indian Premier League (IPL) produces a huge amount of match data every season. This project uses historical IPL delivery-level data to uncover patterns related to:

- Best batsmen performance
- Top wicket-taking bowlers
- Team run statistics
- Over-wise scoring trends
- Dismissal patterns
- Match-level insights

The dashboard is designed for beginners, recruiters, and cricket enthusiasts who want to explore IPL data visually.

---

## 🎯 Objectives

- Identify top run scorers in IPL history  
- Analyze top bowlers by wickets taken  
- Compare team performances based on runs scored  
- Understand scoring trends across overs  
- Explore common dismissal types  
- Build an interactive dashboard using Power BI  

---

## 🛠 Tools & Technologies Used

- **Power BI** – Dashboard creation & data visualization  
- **Python** – Data cleaning and preprocessing  
- **Pandas** – Data manipulation  
- **Google Colab** – Data analysis environment  
- **CSV Dataset** – IPL ball-by-ball dataset  

---

## 📂 Dataset Information

Dataset contains ball-by-ball IPL match data with columns such as:

- `match_id`
- `inning`
- `batting_team`
- `bowling_team`
- `over`
- `ball`
- `batsman`
- `bowler`
- `batsman_runs`
- `total_runs`
- `player_dismissed`
- `dismissal_kind`

---

## 📈 Key Insights

- Top batsmen scored consistently across seasons  
- Certain bowlers dominated with wickets  
- Some teams have significantly higher total runs  
- Death overs generate more scoring opportunities  
- Caught is one of the most common dismissal types
